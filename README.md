# AddHeader

## Config

* Change variables `originHeader` and `headerTest` with your header


## Usage

* Move to the directory containing the file <br/>
    ```
    cd [DIRECTORY]
    ```

* Execute the command specifying the extension

    ```
    ./addHead.py java
    ```
